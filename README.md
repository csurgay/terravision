# terravision
Simple javascript spinning globe, like Silicon Graphics' Onyx with Reality Engine.

1. [map.html](https://csurgay.com/terravision/map.html) converts the world map png to textual format to include into the source code and downloads it
2. [first.html](https://csurgay.com/terravsion/first.html) is the first version with the wrong spherical polar transformation
3. [simple.html](https://csurgay.com/terravsion/simple.html) is the correct transformation and plain graphics
4. [fancy.html](https://csurgay.com/terravsion/fancy.html) is the pimped up version of the terravision rotating globe
